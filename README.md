# Ukrainian translation for Oxygen Not Included

Цей переклад ставить собою за мету створити точний та правільний переклад,
без помилок які копіюються з одного існуючого переклада в інший.

Тому деякі терміни та назви будуть відрізнятися від того що ви можливо бачили
в інших перекладах.

Неофіційна назва нашого перекладу: "Переклад номер 15".
Дивись: https://xkcd.com/927/
